# List of Options

## Dice

- D4
- D6
- D10
- D12
- D20

## Non-Dice Options

- Name from list
- Random number from range (max 1,000,000)
- Yes/No
- Rock, Paper, Scissors
- Rock, Paper, Scissors, Lizard, Spock

# Layout Options

## Glance Options
(What is displayed when the glance is brought into view)

- One option from non-dice options
- Any combination of up to 3 dice

## App Options

(What is displayed when the main app is opened)

- Up to 4 data fields, can be anything
- Re-roll option
